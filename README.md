# Telegram Memory Bot

Простой Telegram-бот: последовательно показывает сообщения и кнопки.

## Как запустить (Replit)

1. Импортировать репозиторий в Replit (Create -> Import from GitHub).
2. В Replit: добавить Secret (Environment Variable) с именем `TELEGRAM_TOKEN` и значением — токен от @BotFather.
3. Открыть Shell и выполнить: `pip install -r requirements.txt`
4. Нажать Run (или в Shell: `python bot.py`).
